# Recipeasy Api

## Get Started
 ```sh
 yarn add gulp -g
 ```

 ```sh
 yarn install
 ```

 ```sh
 yarn start
 ```

## Build
```sh
yarn run build:prod
```

## Test
```sh
yarn test
```
